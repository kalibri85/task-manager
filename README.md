# Task Manager

Simple task manager application built with **Symfony**.  
The app allows users to create, edit, view and delete tasks.

## Features

- Create new tasks
- Edit existing tasks
- Delete tasks
- View task details
- Filter tasks by status
- Hide completed tasks
- Pagination for task list

## Technologies

- PHP 8
- Symfony
- Doctrine ORM
- Twig
- Bootstrap
- KnpPaginatorBundle
- PHPUnit (basic tests)

## Installation

1. Clone the repository
2. Install dependencies (composer install)
3. Configure environment (Edit '.env' file, set your database connection)
4. Create database
5. Run migrations
6. Start the server (symfony server:start)
7. Open in browser: http://localhost:8000
## Tests
Run tests with: php bin/phpunit
